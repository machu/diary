Wikiスタイルを使って定義文にリンクを埋め込むと、表示が変になってしまう。

```
:[[Google|http://www.google.com]]:検索エンジン
```

と書くと、

```
<dl>
<dt><a href="http">Google</a</dt><dd>//www.google.com:検索エンジン</dd>
</dl>
```

こんなhtmlが出力される。 どうやら http: の : で定義文が切れていると判断しているっぽい。
