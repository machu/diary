CHANGELOG をみると 1.2 では文字コードが UTF-8 前提らしい。

```
* Set $KCODE to 'u' by default to enable the multibyte safe String#chars proxy. [Koz]
```

確認してみる。

```
$ rails1.2.1/script/console
Loading development environment.
>> RAILS_GEM_VERSION
=> "1.2.1"
>> $KCODE
=> "UTF8"
$ rails1.1.6/script/console
>> RAILS_GEM_VERSION
=> "1.1.6"
>> $KCODE
=> "NONE"
```
