TextDrive が Rails 1.2 にならないので、 VMware で動かしている Rails を 1.2.2 から 1.1.6 に戻した。

```
$ rails --version
Rails 1.2.2

$ sudo gem uninstall rails --version '>1.2.0'
Successfully uninstalled rails version 1.2.2

$ rails --version
Rails 1.1.6
```
